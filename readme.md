# Crash Simulator

Jednoduchý fyzikální simulátor autonehody vytvořený v Pythonu pomocí Tkinteru.

Simulace počítá:
- reakční dráhu řidiče
- brzdnou dráhu
- rychlost při nárazu
- G-force a riziko zranění
- vliv hmotnosti auta a počtu pasažérů
